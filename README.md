# lazy-image-diary

This is an application to make diary on Evernote.
You can do it by only uploading images.

## Technologies

### Back-end

- [Express.js](https://expressjs.com/)
- [Evernote API](https://dev.evernote.com/doc/start/javascript.php)

### Front-end

- [TypeScript](https://www.typescriptlang.org/)
- [React](https://ja.reactjs.org/)
- [Redux](https://react-redux.js.org/)
- [Jest](https://jestjs.io/ja/)
